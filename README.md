# Mert Şengün — Kişisel Portfolio

Kişisel portfolio web sitesi. Projelerimi, teknik becerilerimi ve sunduğum hizmetleri sergiler.

## Özellikler

- Koyu / Açık tema geçişi (tercih localStorage'a kaydedilir)
- Türkçe / İngilizce dil desteği
- Proje kartlarında CSS animasyonlu önizleme
- Formspree entegrasyonlu iletişim formu (mail doğrudan gelen kutusuna gelir)
- Mobil uyumlu (responsive) tasarım
- Scroll animasyonları (Intersection Observer)

## Projeler

| Proje | Teknoloji | Açıklama |
|---|---|---|
| AlgoTube | HTML/CSS/JS | Algoritma görselleştirme platformu |
| MuglaCargo | HTML/CSS/JS/LocalStorage | Kargo lojistik yönetim sistemi |
| Snake AI | Python/Pygame/Q-Learning | Yapay zeka ile Snake oyunu |
| Tobacco Survey DB | Go/MySQL/HTML/CSS/JS | Veritabanı yönetim uygulaması |

## Kullanılan Teknolojiler

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter)
- Formspree (iletişim formu)
- CSS custom properties (tema sistemi)
- Intersection Observer API (animasyonlar)

## Kurulum

Bağımlılık yok. Doğrudan tarayıcıda aç:

```
index.html dosyasına çift tıkla
```

veya VS Code Live Server ile çalıştır.
